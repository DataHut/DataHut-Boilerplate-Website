Welcome to the DataHut Boilerplate Website.

This project is for a boilerplate website that DataHut uses to create it's web content. We've made this to have a starting point for new projects and to speed up development time. Most of the stuff in here is compiled from other people's projects. We do not claim to have invented anything, just compiled the work of the Internet's best geniuses into a template that we want to use.
